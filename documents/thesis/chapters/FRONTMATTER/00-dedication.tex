Not really typical for a six month master thesis but if you want you can dedicate the thesis to someone here is the opportunity. 

Just keep a few things in mind:

\begin{itemize}
    
    \item Would you be doing something more important with a higher quality you can dedicate later on to the same person? If so don't waste this opportunity!
    
    \item Would this be your last chance to dedicate something to someone? If so go through all the possible options and chose wisely!
    
    \item How would you feel if someone dedicates a work with the same standard and quality to you? 
    
\end{itemize}
